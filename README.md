# AR Edge Explorer

Browser demo of real-time CV modes using OpenCV.js.
Features: Canny, Sobel, LoG, DoG, ORB features, depth-like gradient and segmentation. Works in browser (GitHub Pages).

## How to run
1. Open index.html on GitHub Pages (enable Pages in repo Settings).
2. Allow camera access.
3. Select mode from dropdown.

## Files
- index.html, css/, js/
- report-outline.md (report skeleton)
- statement.md (problem statement)
